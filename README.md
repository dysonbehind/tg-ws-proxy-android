# TgWsProxy Dyson Mobile

**Мобильная реализация MTProto WebSocket прокси. Аппаратно-ускоренный клиент для обхода ограничений Telegram на Android.**

Ускорение Telegram, обход блокировок и полная фоновая автономность.

> **TgWsProxy Dyson Mobile** — это высокопроизводительный графический интерфейс (GUI) и система оркестрации для локального обхода систем глубокого анализа пакетов (DPI) на мобильных устройствах.
>
> В отличие от классических VPN, утилита не пересылает ваш трафик на удаленные серверы. Фрагментация и модификация пакетов происходят локально внутри песочницы приложения. Такой подход позволяет обходить ограничения мессенджера, сохраняя 100% пропускной способности вашего мобильного интернета или Wi-Fi и исходный пинг.

> [!WARNING]
> ### ФОНОВАЯ РАБОТА И БАТАРЕЯ
> Программа использует `Foreground Service` и уведомление для поддержания активного туннеля. Для стабильной работы **ОБЯЗАТЕЛЬНО** добавьте приложение в исключения контроля активности (Battery Optimization -> Don't Optimize). 
>
> Это легитимный инструмент для работы прокси. Без этого разрешения Android закроет приложение через 2-3 минуты после выключения экрана.

> [!IMPORTANT]
> ### STEALTH MODE (Минимизация)
> В версии v1.0.5 добавлена кнопка **Minimize**. Она переводит приложение в режим "Картинка в картинке" (PiP). В этом режиме Android считает процесс приоритетным и не выгружает его из памяти даже при критической нагрузке, что гарантирует 100% аптайм вашего прокси.

## 🔥 Особенности сборки v1.0.5

- **ГИБРИДНОЕ ЯДРО (C++ & Python)**
  * **Chaquopy Engine:** Логика обхода работает на Python 3.11 внутри Android, обеспечивая гибкость настроек.
  * **Native Binaries:** Криптография и пакетная обработка выполняются на скомпилированном C++, что исключает задержки (Lag-free).

- **ИНТЕГРАЦИЯ С TELEGRAM**
  * **One-Tap Setup:** Кнопка "Open Telegram" автоматически прописывает прокси `127.0.0.1:1080` в мессенджер. Больше не нужно вводить данные вручную.
  * **Live Uptime:** Анимированный график аптайма на главном экране показывает реальное время работы службы.

- **ОПТИМИЗАЦИЯ ТРАФИКА**
  * **Smart Fragmentation:** Динамическое разделение TCP-потока для обхода самых агрессивных систем DPI.
  * **Low Data Mode:** Минимальное потребление ресурсов системы и трафика в режиме ожидания.

## 🚀 Архитектура и запуск

Проект полностью избавляет от необходимости использовать сложные настройки или сторонние приложения.

* **Технологический стек:** Kotlin (UI), Jetpack Compose, Chaquopy (Python Engine), C++ Core.
* **Изоляция:** Приложение создает локальный сокет, который доступен только вашему устройству. Никакой утечки данных во внешнюю сеть.
* **Приоритет:** Процесс работает с флагом `FOREGROUND_SERVICE_TYPE_DATA_SYNC`, что позволяет обрабатывать пакеты с минимальной задержкой.

## ⚙️ Использование

1. **Запуск:**
   - Нажмите кнопку **START** на главном экране.
   - Дождитесь, пока статус сменится на **ACTIVE**.
2. **Настройка мессенджера:**
   - Нажмите **COPY LINK** или **OPEN TELEGRAM**.
   - Подтвердите добавление прокси в открывшемся окне Telegram.
3. **Фоновый режим:**
   - Нажмите **MINIMIZE** для перехода в режим "невидимки" (PiP).
   - Теперь можно пользоваться телефоном как обычно — прокси будет работать в фоне.

## 📌 Принципы продукта
