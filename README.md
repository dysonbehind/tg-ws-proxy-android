**Мобильная реализация MTProto WebSocket прокси. Аппаратно-ускоренный клиент для обхода ограничений Telegram на Android.**

Ускорение Telegram, обход блокировок и полная фоновая автономность.

> **avocado** — это высокопроизводительный графический интерфейс (GUI) и система оркестрации для локального обхода систем глубокого анализа пакетов (DPI) на мобильных устройствах.
>
> В отличие от классических VPN, утилита не пересылает ваш трафик на удаленные серверы. Фрагментация и модификация пакетов происходят локально. Такой подход позволяет обходить ограничения мессенджера, сохраняя 100% пропускной способности и исходный пинг.

> [!CAUTION]
> ### ПОДДЕЛКИ

> [!WARNING]
> ### ФОНОВАЯ РАБОТА И БАТАРЕЯ
> Программа использует `Foreground Service` и `WakeLock`. Для стабильной работы **ОБЯЗАТЕЛЬНО** добавьте приложение в исключения контроля активности (Battery Optimization -> Don't Optimize). Без этого Android закроет прокси через пару минут после выключения экрана.

> [!IMPORTANT]
> ### 🧠 ИНТЕЛЛЕКТУАЛЬНЫЙ РЕЖИМ PiP (Stealth Mode)
> Режим "Картинка в картинке" (PiP) работает автоматически. При запущенном прокси просто сверните приложение кнопкой "Домой" — оно сожмется до минимума, не мешая работе и гарантируя 100% аптайм.

## 🚀 Архитектура

* **Технологический стек:** Kotlin (UI), Jetpack Compose, Chaquopy (Python 3.11 Engine), C++ Core.
* **Изоляция:** Локальный сокет доступен только вашему устройству. Никакой утечки данных.
* **Приоритет:** Процесс `FOREGROUND_SERVICE_TYPE_DATA_SYNC` минимизирует задержки обработки пакетов.

## ⚙️ Использование

1. **Запуск:** Нажмите кнопку **START**. Статус должен смениться на **ВКЛЮЧЕНО**.
2. **Настройка:** Нажмите **OPEN TELEGRAM**. Прокси `127.0.0.1:1443` пропишется автоматически.
3. **Фон:** Просто сверните приложение. Режим PiP активируется сам, поддерживая связь.
